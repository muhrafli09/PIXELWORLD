Drop custom fonts in this folder to activate UI font presets.

Supported gameplay font filenames (first found is used):
- PixelOperator.ttf
- pixel_operator.ttf
- m6x11.ttf
- Dogica.ttf

Supported menu font filenames (first found is used):
- Nunito-Regular.ttf
- Nunito.ttf

Notes for pixel fonts (Godot import settings):
- Filter: Off
- Mipmaps: Off
